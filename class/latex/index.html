Introduction

Introduction

  • LaTeX: Typesetting system for scientific and mathematical documents.
  • Enables authors to typeset and print with top typographical quality.
  • Pronounced "Lay-tech"; uses TeX formatter as its engine.
  • TeX: Program by Donald Knuth for typesetting text and formulas.

Why LaTeX?

  • Easy to use, especially for typing mathematical formulae
  • Portability across Windows, Unix, and Mac operating systems
  • Stability and interchangeability

Why LaTeX? (continued)

  • High-quality output
  • Most journals have their LaTeX styles
  • You may be compelled to use it, as it's widely adopted

Why LaTeX? (continued)

  • Documentation and forums provide ample support
  • Universally accepted among researchers
  • Error finding and troubleshooting are not difficult

Mathematical symbols in LaTeX

Math Mode in LaTeX

Math mode in LaTeX allows you to include mathematical expressions and equations in your document.

Inline Math Mode: 
$...$

In inline math mode, you can enclose mathematical expressions within dollar signs ($). For example:

The quadratic formula is $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$.

Use inline math mode for short mathematical expressions within text paragraphs.

Display Math Mode

In display math mode, you can enclose mathematical expressions within 

\(...\) or \[...\]
For example:

\[
    \int_{0}^{\infty} e^{-x^2} \, dx = \frac{\sqrt{\pi}}{2}
\]

Use display math mode for standalone mathematical expressions or equations that should be displayed separately from the surrounding text.

Bibliography in LaTeX

Bibliography in LaTeX

To include a bibliography in LaTeX:

  1. Prepare a BibTeX file (.bib) with your references.
  2. Insert citations in your document using \cite.
  3. Specify the BibTeX file and citation style with \bibliography and \bibliographystyle.

Bibliography in LaTeX


% In your LaTeX document:
\documentclass{article}

\begin{document}

Some text \cite{example}.

\bibliographystyle{plain}
\bibliography{references}

\end{document}

% In references.bib (BibTeX file):
@article{example,
  author = {Author, A.},
  title = {Title of the Article},
  journal = {Journal Name},
  year = {2022},
  volume = {1},
  pages = {1--10}
}

Introduction to Beamer

Beamer is a LaTeX document class used for creating presentations.

Features of Beamer

  • Professional-quality presentations
  • Support for overlays and animations
  • Easy integration with LaTeX
  • Customizable themes and styles
