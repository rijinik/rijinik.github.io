Operators in Python

Operators in Python are special symbols or keywords that are used to perform operations on variables and values.

Arithmetic Operators

Used to perform basic mathematical operations like addition, subtraction, multiplication, and division.


x = 10
y = 3
	
print(x + y)  # Addition: 13
print(x - y)  # Subtraction: 7
print(x * y)  # Multiplication: 30
print(x / y)  # Division: 3.333...
print(x % y)  # Modulus: 1
print(x ** y)  # Exponentiation: 1000
print(x // y)  # Floor Division: 3

Comparison Operators

Used to compare two values. They return either True or False.


x = 10
y = 3
	
print(x == y)  # Equal to: False
print(x != y)  # Not equal to: True
print(x > y)   # Greater than: True
print(x < y)   # Less than: False
print(x >= y)  # Greater than or equal to: True
print(x <= y)  # Less than or equal to: False

Logical Operators

Used to combine conditional statements.


x = True
y = False
	
print(x and y)  # Logical AND: False
print(x or y)   # Logical OR: True
print(not x)    # Logical NOT: False

Bitwise Operators

Used to perform bit-level operations on integers.


x = 10  # 1010 in binary
y = 3   # 0011 in binary
	
print(x & y)  # Bitwise AND: 2 (0010 in binary)
print(x | y)  # Bitwise OR: 11 (1011 in binary)
print(x ^ y)  # Bitwise XOR: 9 (1001 in binary)
print(~x)     # Bitwise NOT: -11 (inverts all bits)
print(x << 2) # Bitwise left shift: 40 (101000 in binary)
print(x >> 2) # Bitwise right shift: 2 (0010 in binary)

Assignment Operators

Used to assign values to variables.


x = 10  # Assignment: x is 10
x += 3  # Add and assign: x is 13
x -= 3  # Subtract and assign: x is 10
x *= 3  # Multiply and assign: x is 30
x /= 3  # Divide and assign: x is 10.0
x %= 3  # Modulus and assign: x is 1.0
x //= 3 # Floor divide and assign: x is 0.0
x **= 3 # Exponentiate and assign: x is 0.0
x &= 3  # Bitwise AND and assign
x |= 3  # Bitwise OR and assign
x ^= 3  # Bitwise XOR and assign
x >>= 3 # Bitwise right shift and assign
x <<= 3 # Bitwise left shift and assign

Special Operators

  • is and is not are identity operators.
  • in and not in are membership operators.

x = [1, 2, 3]
y = [1, 2, 3]
z = x
	
print(x is z)       # True because z is the same object as x
print(x is y)       # False because x and y are not the same object
print(x == y)       # True because x and y have the same content
	
print(2 in x)       # True because 2 is in the list x
print(4 not in x)   # True because 4 is not in the list x

Python input() Function

The input() function in Python is used to take user input from the console.

Reading a String

By default, input() reads input as a string.


# Reading a string
string_input = input("Enter a string: ")

print(string_input)

Reading an Integer

To read an integer from the user, you need to use the int() function to convert the input string to an integer.


# Reading an integer
integer_input = int(input("Enter an integer: "))

print(integer_input)

Reading a Float

To read a floating-point number (float) from the user, you use the float() function to convert the input string.


# Reading a float
float_input = float(input("Enter a float: "))

print(float_input)

Modules

Modules

  • Modules are files containing Python code that can be imported and used in other Python programs.
  • A module can contain functions, classes, and variables.
  • You import a module using the import keyword, followed by the module name.

math Module


import math

num = 16
square_root = math.sqrt(num)

You can also import specific functions or variables from a module:


from math import sqrt

num = 25

Formatting Text for Output

Formatting Text for Output

  1. Using the % operator.
  2. Using the {} (format) method.
    • It’s generally recommended to use {} format method.

Tuples

Immutable collections of items

Mutability and Tuples


	# Tuples are immutable
	t = (1, 2, 3)
	try:
		t[0] = 4
	except TypeError as e:
		print(e)  # 'tuple' object does not support item assignment

Tuple Assignment


	# Tuple assignment
	a, b = 1, 2
	print(a, b)  # 1 2

Tuples as Return Values


	# Tuples as return values
	def min_max(numbers):
		return min(numbers), max(numbers)
	
	print(min_max([1, 2, 3, 4, 5]))  # (1, 5)

Tuple Operations


	# Tuple operations
	t1 = (1, 2, 3)
	t2 = (4, 5, 6)
	t3 = t1 + t2
	print(t3)  # (1, 2, 3, 4, 5, 6)

Functions, Modules, and Packages

Creating reusable code

Function Definition

Parameters and Arguments

Lambda Functions


	# Lambda functions
	add = lambda x, y: x + y
	print(add(2, 3))  # 5

Recursion


	# Recursion
	def factorial(n):
		if n == 0:
			return 1
		else:
			return n * factorial(n - 1)
	
	print(factorial(5))  # 120

Modules and Packages


	# Importing modules and creating packages
	import math
	print(math.sqrt(16))  # 4.0
	
	# Creating a module
	# Save this as mymodule.py
	def greet(name):
		return f"Hello, {name}!"
	
	# Importing from a module
	from mymodule import greet
	print(greet("Alice"))
	
	# Creating a package
	# mypackage/__init__.py
	# mypackage/mymodule.py
	# Importing from a package
	from mypackage.mymodule import greet
